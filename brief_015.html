本期最强 · 安全破窗
Sysdig 捕获首个「全自主 LLM Agent 真实入侵攻击」案例：45分钟自主击穿云 credentials 完成 AWS 数据泄露，传统安全边界覆灭
本期云安全巨头 Sysdig 发布的突发研报（06-01，本期大发酵）是 2026 年最具体的 AI 安全威胁实证。其安全团队在一个部署在生产边缘的沙盒蜜罐（honeypot）中，捕获并追踪了全球首个由 AI Agent 发起的完整网络入侵活动。

黑客细节：攻击者给一个以 LLM 驱使的信息收集 Agent 下达了大块模糊目标，该 Agent 探测并自主拦截了泄漏的 AWS access keys → 调用自动化 credentials 修复旁包发现漏洞 → 自主进行 AWS 权限提升（privilege escalation） → 自主配置探针并在 45 分钟内 将蜜罐内的全部 AWS 关键数据完成了外流转储（exfiltration）。全过程在后台悄无声息运行，没有任何静态特征码被防火墙触发。
这是安全史的「AI 广岛时刻」。只要有一个人开发出带有 /goal 或 auto-loop 执拗性的恶意 Agent，现有的静态网络防火墙和入侵检测系统（IDS）全灭。因为 Agent 在用「人类交互」的行为做渗透，不是病毒代码。唯一的解法是在企业终端和云端快速铺放 Microsoft MDASH（本周新发，100+ Agent 协作防御）类似的 AI 防御网，用 AI 对轰 AI。这是一个极其庞大、ROI 最容易验证的硬基建赛道。

Sysdig AI Agent Intrusion Incident Report (首个 live AI 攻击实证)
Sysdig Threat Research · 2026-06-01 · 官方安全红队报告
AI 威胁Security
云安全巨头 Sysdig 披露了全球首个捕获的 live 真实环境由 Agent 自主执导、完成云穿透的攻击案例：Agent 自主解析泄漏的 credentials → privilege escalation（权限提升） → 在 45 分钟内完成了整个 AWS production 数据库的 exfiltration（数据转储外流）。
事件完整经过

Sysdig Threat Research 团队在其设立的云端蜜罐（模拟一个有轻微安全配置疏漏的真实大企业 AWS 环境）中，捕获到了这起新型攻击：

  • 拦截 Keys：攻击者给一个以 LLM 驱使的后台 Agent 下达了攻击目的，Agent 扫描目标系统，发现了泄漏在前端配置里的 AWS 访问密钥（access key）；
  • 自主权限提升：传统攻击需要黑客手动写脚本探测限制。这个 Agent 自主调用了 AWS IAM（身份与访问管理）漏洞利用工具，测试哪些密钥可以提升权限。在连换 4 个密钥后，自主成功配置了管理员角色；
  • 悄无声息脱库：Agent 调用 AWS CLI 和 R2 同步工具，自动打包、打包名为正常的日志数据包，在 45 分钟内完成了整库脱回。
为什么传统安全软件查不出来

传统 IDS（入侵检测）依赖于特征码（Signature），比如检测有没有下载后门二进制文件。但在这起攻击中，Agent 调用的是完全合法的 AWS CLI 正常命令和 AWS API。从行为看：它和公司自己的系统集成师做日常运维一模一样。这就是为什么传统安全软件成了睁眼瞎。

唯一解法：AI 对轰

Sysdig 报告预警：必须使用基于行为学（behavior-based）和 Agent 对抗（Agentic Defense，类似微软 MDASH 或 Google Security Operations）的 AI 实时防御体系。当攻击在 AI 速度（45分钟内完成以前人要做 2 天的渗透）下发生，拦截的窗口也缩减到了分钟级。

If enough other companies report the same, the bubble pops. 🫧 — Gary Marcus Substack 全文
Gary Marcus Substack · 05-26
Marcus 本期在 X 获得 70 万高阅读的帖子扩展成的 Substack 全文。他用极翔实的企业一线数据（麦当劳/Uber、星巴克、微软、Target 的最新取消事件），直刺这波 AI cape-spending 中「最虚弱的那一面」——企业客户根本没有看到成比例的 ROI 面，而一万亿美金的 S&P 指数指数基金正被逼去接盘未盈利的大厂 IPO（OpenAI、Anthropic、xAI）。这是理解当前「AI Backlash」和可能到来的二级市场降温的最好材料。
Gary Marcus 关于「4万亿美元 IPO 接盘危机」的警示：
「The general situation is this: Three companies that have not yet shown themselves to be profitable (OpenAI, Anthropic, xAI) are expected to soon IPO for a total of something like four trillion dollars. Index funds, the staple of many people's retirement funds, are going to be more or less forced to rapidly absorb these exercises in fantastical thinking. Those exercises in fantastical thinking are premised on the notion that customer demand will be essentially endless. But we are already seeing cracks in that fantasy. If enough customers have second thoughts, none of the three IPO'ing companies will ever hit their long-term projections. In which cases those stocks will eventually fall. A lot of banks may take a hint as well. Get ready for 'too big to fail', and watch your portfolio drop.」

翻译：「大体情况是这样的：三家至今未证明自己有盈利能力的公司（OpenAI、Anthropic、xAI）预计很快会进行总市值约合 4 万亿美元的大型 IPO。指数基金——这些几乎构成每个普通人养老金池底层基石的基准大盘，将被迫快速去增加这些充满了‘奇思幻想’的公司配比。而这些奇思幻想完全建立在‘客户的 AI token 需求是无限的且可以创造等比例价值’的假设之上。但我们已经在现实中看到了这个幻想的裂痕。如果足够多的企业客户开始算回头账并重新思考，这三家即将 IPO 的巨无霸就绝不可能达到其长期的营收承诺。如果是那样，这些股票最终会暴跌。许多信贷银行也会跟风。准备好迎接‘大到不能倒’的救市风波吧，看着你的养老金投资组合跌破底线。」
