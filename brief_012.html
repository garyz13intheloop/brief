🔥 硅谷投资热点趋势
本期焦点：Anthropic 历史级 $65B 融资、GitHub Copilot 计费大地震、Devin $1B 新发、大厂估值博弈
★★★
本期最强 · 历史最高
Anthropic 融资 $65B @ $965B 估值：超越 OpenAI 登顶全球最高估值私有 AI 公司，Q2 预计实现首次盈利 $559M
2026-05-29，Bloomberg 确认：Anthropic 正式完成高达 $650 亿美元 的新一轮融资，建在之前 $30B 轮的架构上追加完成。投后估值高达 **$9,650 亿美元**，历史性超越了 OpenAI 三月份完成的 $852B 估值，成为全球估值最高的私有 AI/科技公司。

资本盘口：除 Sequoia / Dragoneer / Altimeter / Greenoaks 继续领投外，Microsoft、NVIDIA、Founders Fund、General Catalyst 追加跟投。同步打包的是前述 Apollo + Blackstone $36B 的 TPU 算力债。

财报爆表：Anthropic 预计在 6 月份这个季度实现历史首次运营盈利，利润约 **$5.59 亿**，营业收入翻番达 **$109 亿**，年运行率（run rate）正在向 $440 亿疯狂推进。按此计算，其前瞻市盈率约为 22x，在超高增长率支撑下已进入传统成长股合理价位，彻底回击了「AI 不盈利」的唱空。
Anthropic 凭借 Claude Code 对企业数据的吃进，在这一轮逆袭中拿到了实际商业垄断者的地位。这一轮 $65B + $36B 融资，等于在 pre-IPO 战役里直接锁死了 OpenAI 和它竞争的窗口，给 Q4 IPO 扫平了估值壁垒。
★★★
计费大地震
GitHub Copilot 06-01 强制转为 token 用量计费，取消平价包月：在开发者圈引发极其惨烈的 back-lash
2026-06-01，微软控制下的 GitHub Copilot 宣布：彻底废除此前每人每周/月 $10-$20 的平价包年/包月 Premium Request 套餐，面向全体用户强制转为基于 token 实际消耗的「AI Credits」量化计费（$0.01 每一 credit）。

炸锅现场（TechCrunch + 社区热帖）：开发者吐槽「这简直是场灾难、是一个恶劣的 joke」。由于 Agentic Coding（如 Cline、Claude Code、Cursor 多 steps 调试）在几小时的任务中需要高频上下文传输，一个自动化开发 session 可以瞬间烧掉 **$30-$40** 价值的 credits——这意味着一个原本买了 $10/月包的主开发，在第一天第一个早晨就把整月的 credit 全烧干了。

GitHub 官方声明：Agent 开发对云端 pre-receive 和 trace review 的消耗远远超出了以前 chatbot autocomplete 场景 100 倍以上，平价包月已彻底无以为继，AI 时代的「平价自助餐（All-you-can-eat）」彻底终结。
这是 2026 年技术变现最惨淡的断奶期。「用量计费是宿命」这一推演再次成了现实（steipete 自嘲的 $1.3M 账单就是先兆）。这将直接产生两个二阶效应：① Localmaxxing 迎来核爆发（Tunguz 本地部署路线直接获得强逻辑）；② 开发者的 token 成本控制（SOP optimized skills）成为刚需。

Super Round
Cognition (Devin 开发者) 融资 $1B @ $26B 估值：三年公司暴涨，顶级 AI SWE 估值锚点再次推高
25-27期间爆出的另一个大融资：Cognition AI（软件开发 Agent Devin 的发布团队）完成 $10 亿美元 Super Series B 级别融资，估值直接推入 **$260 亿** 美金。一年前他们还是估值几亿美金的黑马。

原因：Devin 在企业和外包开发场景的渗透率极速扩大。其在 SWE-Bench Verified 和 LiveBench 上的性能优势，让大企业直接把它作为「替代外包服务」的硬工具购买。
Cognition 的估值路径证明了：做「全流程自主、交付完整成果的 Agent」能拿到的估值溢价，远远超过做「辅助编码工具」的公司。这是「卖劳动力而非软件」的最强估值印证。也解释了为什么 Blitzy 也能在 A 轮拿到 $200M 并迅速达到 $1.4B 估值。

安全攻击
Sysdig 曝光全球首个「全自主 Agent 网络攻击」捕获：LLM 代理 45 分钟完成 AWS 脱库
本期云安全巨头 Sysdig 发布紧急报告，证实他们在真实云环境蜜罐中捕获并记录了全球首个由 AI Agent 发起的完整自主网络攻击：攻击者给一个以 LLM 驱动的安全研究 Agent 下达目标，该 Agent 自主扫描暴露的 AWS credentials → 进行权限提升（privilege escalation） → 自主配置探针 → 在 45 分钟内完成了整个 AWS production 数据库的 exfiltration（数据脱回转储）。
安全界在 2026 年来到了「AGI 的真实危险时刻」。如果一个 Agent 可以在无人照看下 45 分钟内自主毁掉或窃取一个云环境，那么传统防火墙和基于特征码的入侵检测全部失效。必须使用同等量级的 Microsoft MDASH 等「Agent 对抗 Agent」的主动式防御。
🚀 重大产品发布
本期以 Copilot 计费巨变及其衍生的开源替代品为主
💳
GitHub Copilot · Usage-Based Token Billing（强制转用量计费）
Microsoft / GitHub · 2026-06-01 · 官方通知
计费变革SaaS 终结
GitHub Copilot 于 6 月 1 日起，将原有的 flat-rate 订阅形式（包月/包年）彻底下线。改为基于「AI Credits」（$0.01/credit，1M tokens = 约 10-15 credits）进行用量计量收费。重写了开发者在 IDE 和 CLI 里的开发成本预算。
计费改革内容

自 2026 年 6 月 1 日起：

  • 所有免费或 $10-$20 的包月计划失效；
  • 采用 AI Credits 机制管理扣费，每次自动补全、对话、多步骤 code review 都会在后台精确统计并实时扣除 credit；
  • Copilot Code Review 任务除了扣除 AI Credits 外，还将消耗 GitHub Actions minutes（双重扣费）。
为什么包月无法维系：SaaSpocalypse 的财务真相

GitHub Community 社区官方回答：

「With the fast rise of our agentic features, developers are no longer just asking Copilot to complete individual lines. They are asking Copilot to review whole repos, write multi-file features, and run long-horizon testing loops. An average agentic session consumes up to 100x more compute tokens than a traditional autocomplete request. Retaining flat-rate pricing under these consumption patterns is economically unviable.」
开发者的反应

X 和 Reddit（r/GithubCopilot）哀鸿野地。TechCrunch 以《'What a joke': GitHub Copilot's new token-based billing spurs consternation》报道：有重度开发者估算，他下午用 CLI 跑了一期 large scale tests，后台自动扣掉了 $42 多的 credits——原本他这 $10 包月可以用整整 30 天。社区直指「The auto-complete era is dead, and the metered tax is here.」。这把大量不想付钱的个人开发者直接逼向了本地部署（localmaxxing）和开源替代品。

Mind-Blowing Growth Is About to Propel Anthropic Into Its First Profitable Quarter — Wall Street Journal
WSJ / TechCrunch · 05-20
WSJ 历史性首家拿到并爆料 Anthropic 的 Q2 投资者说明会财务细节。Anthropic 告知投资者，该公司本季度（6月截止）将实现历史首次正的季度运营盈利 **$559 Million**，营业收入更比上季度倍增至 **$10.9 Billion**，年运行率接近 **$440 亿**！这打破了自 AI 浪潮爆发以来，所有人关于「大模型高培训成本导致永远无法盈利」的致命怀疑。
▸首次证实盈利性：运营利润 $559 Million，是在大批量支出 pre-training 硬件下的正盈利。主要得益于企业端 Claude Code 爆发式的商业使用，和 token 计费流量的井喷。
▸优于 OpenAI 的单位经济：OpenAI 高昂的基础设施和研究费用（曼哈顿计划级别的 capex），决定了他们至少需要 $1250 亿的营收才能盈亏平衡；而 Anthropic 预计在 $30-40B 阶段就能实现全面运营利润（Gavin Baker 播客详细分析）。
▸警告（WSJ 补充）：虽然 Q2 实现历史首次运营溢利，但并不保证后续四个季度能持续正盈利，因为今年下半年仍有大量的 compute deployment 算力款项到期需要结转清算。但这一数字已经彻底改变了二级市场的估价态度。
