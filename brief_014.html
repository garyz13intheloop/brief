Cognition (Devin 开发者) 融资 $1B @ $26B 估值：自主交付型（Service-as-a-Software）Agent 拿走应用层最高估值溢价
2026 年 6 月初，AI 程序员 Devin 的发布团队 Cognition AI 完成新一轮高达 $10 亿美金 的 B+ 轮（或超级战略轮）融资，投后估值达到了惊人的 **$260 亿美元**。这一估值距离他们去年初首次亮相、引爆 AI 程序员炒作时，暴涨了将近 50 倍。

估值基础支撑：不仅仅是炒作。Devin 在大企业（如 Rakuten、TCS）和跨国外包（Infosys 等）的实际生产整合指标非常高。大企业将其作为「可自主交付成果、通过 API 安全托管、按成果付费」的数字员工直接部署，大笔替换掉了传统的人工后端外包开发。这种「卖劳动力而非软件（Selling Labor, Not Software）」的单位经济，在财务层面对接了传统外包行业几千亿美金的 TAM。
对于 Creekstone：Devin 的估值爆发再次印证了我们的核心应用层假说——**「AI 会吞噬 SaaS 包装，变成端到端的服务（Service-as-a-Software）」**。只有能自主完成并对 done 负责的 Agent 最值钱。中小型 AI 应用的「提效工具」叙事，在这一千倍的估值差距面前高下立判。
★★★
技术隐喻
fchollet ♥1,262：「Coding Agent 是迷宫里的盲松鼠」——揭示大公司 Agent 部署最核心的数据瓶颈
本期技术领袖 François Chollet 发表了今年在 X 上最被推崇的 AI 开发金句之一：「A mental model for working with coding agents is that they're blind squirrels running into a maze and bumping into walls.」

深层含义（Digg & Tech 社区深度解读）：大家都以为 AI 编码越来越快（10x 代码量），是因为模型变聪明了。但真实的工程现实是：Agent 是没有全局方向感的，它通过试错（bumping walls）来前进。大企业在 Agent 部署中遇到最大的墙：企业内部的历史文档、混乱的 databases、没有及时更新的知识图谱。当 Agent 迷路时，它会疯狂读取过期数据，生成大批废话甚至引入老 bug，彻底陷入「无意义的自动循环」。
限制盲松鼠无序试错的，是「精确定义的约束（Harness）」以及「清洁的数据上下文（a16z 的 Context Layer 论题）」。谁能帮企业把散落在 Slack、Confluence 和旧代码里的「数据上下文」清洗干净喂给 Agent，谁就掌控了 Agentic 时代的数据话语权。

gregisenberg「5天旧金山之行感悟」：硅谷创业文化彻底重写，所有人都在拼抢「垂直行业专有数据集」完成 post-training

没有任何创业公司在聊「我们写什么 App 界面、我们融多少钱」，现在顶级创始人的日记本里只有这几个词：① post-training：怎么拿到某个无聊行业（牙医、卡车合规、法律、保险）的顶级专家私有数据，训练自己的小模型（7B-35B）；② harness engineering：怎么设计确定性规则把盲松鼠框住；③ Localmaxxing：怎么在本地 M5 MacBook/iPhone 上跑出高精度推理，摆脱大厂 $200/Mtok 的 token 暴政。

Genspark Claw · 「AI 员工」原生平台发布
Genspark · 2026-05-25 · $385M 融资同期披露
AI EmployeeB2B
Genspark 正式发布其首个「AI 员工」应用——Genspark Claw，以及配合的 Genspark AI Workspace 3.0。不同于普通的 prompt 生成工具，Claw 允许用户用自然语言向其「delegate outcomes」（下达结果，不只是任务步骤级指令），Claw 自动跨软件界面、执行多步骤任务并返回完毕的数据资产。
Genspark 3 年的成长数据

伴随此次发布，Genspark 透露已在 2026 年 3 月底实现 $200 Million ARR 运行率（B2C AI 搜索 + B2B 会议/财务 Agent 联合），并完成其 Series B 扩展轮（LG Technology Ventures 和日本 SBI 领投，投后市值达 $1.6B，确认新独角兽地位）。

Genspark Claw 的技术核心：向 Outcome 负责

官方发布说明中指出，Claw 配备了自主界面的探针：它不是在没有界面的后台跑 API，而是在真实的软件环境里运行（类似 Codex Computer Use 的 B2C 泛用版本）：

  • 你可以说：「帮我把这个月的发票全部收齐，对照账单，找出金额差异超过 $20 的项目，把结果做成 Excel 放在我桌面上。」——Claw 自动打开邮件，找出附件，自动创建计算 Excel。
  • 在 Workspace 3.0 界面上，用户可以看到 AI 员工的实时工作轨迹，随时点「暂停」或「重试」，就和真实雇人一模一样；
  • 这标志着 B2C AI 公司（此前以 Kimi、Sora 趣味内容为主）向「企业端真生产力（Outcome-driven Agentic Service）」业务的彻底转型。美国 B2B 市场需求正在以每年 3-5x 级别吞噬这类工具。

François Chollet
@fchollet · Keras 作者 · ARC AGI
♥ 1,262
05-26
「A mental model for working with coding agents is that they're blind squirrels running into a maze and bumping into walls.」
极其精妙的技术隐喻。Coding Agent 并不是真懂整个项目的全局软件工程结构。它在工作时是「在限制内做随机漫步并通过撞墙（测试报错、编译错误）来获取反馈前进（bumping walls）」。大厂部署 Agent 遇到最大挫败就在这里：项目本身的数据太乱、没有干净的 context。盲松鼠在不干净的迷宫里会直接撞死。所以，harness（把盲松鼠框住的物理结构）和 Context Layer 质量决定了最终 Agent 是否真的具有高 ROI。

🎙 播客 · Let's Data Science
Demis Hassabis Declares 'Foothills of the Singularity' at I/O — Let's Data Science
LDS · 05-21
LDS 对 Google DeepMind CEO Demis Hassabis 在 I/O 收尾发言「踩在奇点脚下」及 ROWAN CHEUNG 深度专访最系统的复盘。详细拆解了为什么「AI 会在未来几年压缩生命科学 100x 时间」、以及为什么 AI 智能眼镜是未来的 killer app，而不是手机或桌面 App。
▸奇点山脚下（实录引用）：「When we look back at this time, I think we will realize that we were standing in the foothills of the singularity. It will usher in a new golden age of scientific discovery and progress.」
▸生命科学的「100x 速度压缩」：AlphaFold 解决了蛋白质静态结构预测。Isomorphic 的统一 IsoDDE（Isomorphic 药物引擎，本周完成 $2.1B 巨额 B 轮）是真正把算力引入分子活性和抗体自主设计上。Hassabis 预测：「新药候选分子的从靶点识别到确认，未来将从 10 年缩短到 2 周内通过全 AI 模拟完成筛选定位。」这就是 AGI 的价值。
▸为什么眼镜是 AGI-era 理想载体：因为人类在通过眼睛建立和世界的「高密度物理交互」。目前手机的「打开应用 → 摄像头对准目标 → 输入 Prompt」有无法逾越的交互损耗（Friction）；而基于 Android XR + Gemini 2.5 Pro 实时低延迟推理的智能眼镜，能完美做到「你看一眼，AI 就知道了你在看什么并且在后台自主开启任务处理」——这就是第一人称 Agent OS 层的终极生态位。
Hassabis 谈「AI 如何改造医药研发」（半翻译）
「The problem with traditional drug discovery is that it's a brute force game of trial and error in wet labs — we call it 'spray and pray' in a way. You screen 100,000 components in a multi-year loop to hope three connect. Isomorphic is building a unified model, IsoDDE, that does all these screen in silco in real-time. We can predict not just structures, but exact bind affinities and physiological effects of molecules directly on the model. If we can compress the discovery loop of a new oncology candidate from 10 years to 2 weeks, we save billions of dollars, but more importantly, we save millions of lives. That's why Alphabet and Thrive Capital are putting $2.1B in our Series B — this is a sovereign-scale mission.」

翻译：「传统药物研发最大的痛点是，它在湿实验室里几乎是一场纯暴力试错的游戏。你需要在好几年的循环里，大面积测试 10 万种化合物成分，然后祈求有两三种能产生实际的结合。Isomorphic 正在构建的是一套统一的大模型 IsoDDE，在虚拟算力空间里实时预测完成这所有的筛查。我们不仅预测结构，更能直接大模型预测分子的精确结合亲和力（bind affinities）以及它的生理活性效应。如果我们能把一个新的肿瘤候选药物的发现周期从 10 年压缩到 2 个星期内，这拯救的是数十亿美元的资金开销，但更重要的，是拯救数百万个正在死去的生命。这也是为什么 Alphabet 和 Thrive Capital 会把 $21B 的巨款在 B 轮砸向我们——这是一个主权级别的使命。」
