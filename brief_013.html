🔥 硅谷投资热点趋势
本期：微软 7 自研模型发布（MAI-Thinking-1）、对 OpenAI 依赖削弱叙事、Copilot 计费发酵
★★★
本期最强 · 微软 Build 2026
微软发布 7 个 in-house AI 模型，首推 reasoning 级 MAI-Thinking-1：巨头「自主、独立、去 OpenAI 化」的最强动作
2026-06-02，微软在其年度 Build 开发者大会上扔出重磅核弹：宣布推出由其 Microsoft AI Superintelligence 团队完全自主研发的 7 个 in-house AI 基础模型系列，其中包括其首个自主推理模型：MAI-Thinking-1。

CNBC、Yahoo Finance 头版报道：这是目前为止微软试图「Lessen reliance on OpenAI and lower costs for developers」的最清晰、最决裂性质的手腕。虽然微软持有 OpenAI 约 49% 的利润分成，但过去一年大模型超限 capex 重压和 OpenAI 的极高 API 溢价，逼着微软在内部完成自主替代。

技术定位：MAI-Thinking-1 定位为「midsize reasoning model designed for high efficiency and low-token cost」，用极高性价比打入原本由 GPT-4.5/5.4 统治的高级推理场景。
对于早期 AI 应用和 Agent 团队：大厂的 AI 策略完成了从「大笔资助模型 startup」到「建立完全自主模型线」的最终滑坡。微软的 7 自研模型上线，意味着 OpenAI 失去了最绝对和排他的 runtime 分发保护罩——这是一个深刻的地缘格局断裂。对垂直 Agent：应该加速「多模型对冲」架构配置（不把瓶颈压在单一 API 供应上）。

商业辩论
Copilot 切换至用量扣点，SsaSpocalypse 的财务结算红利到达：AI 模型公司正在通过「收窄平价空间」实现历史首次盈利
本周 GitHub Copilot 正式转入 AI Credits 扣费（06-01），在开发者圈子内产生强震荡。结合同周期 Wall Street Journal 独家爆料的「Anthropic 将在 Q2 取得历史首次正运营利润 $559 Million」这一震撼财务数据，两者指出了同一个行业大冰层开裂：

基础模型公司和平台商（微软/GitHub、Anthropic、OpenAI）开始正式**拒绝为用户的超额 token 消耗买单**。通过把 flat-rate 砍掉、推行精确用量扣点、或大幅拉高 premium license 价格，他们开始进入真正可以把 token 转化为实际运营利润（operating profit）的健康账面阶段。
「AI 烧钱不盈利」的怀疑论被账面利润回击。但二阶代价是：中小型开发者和中轻度 Agent 项目将承载真实的成本压力。这会让「本地运行中模型」（localmaxxing）和「开源 Agent harness 优化（减少 token 浪费的 code Purge 技巧，如 steipete 本期提示）」从极客技巧变成小企业求生的绝对刚需。

大厂裂痕
OpenAI 准备起诉 Apple 的二阶讨论发酵（All-In）：Apple 的硬件特权让模型公司彻底成了「免费打工仔」
All-In 最新期（05-22）由 Marc Benioff 客座，深度剖析了 OpenAI 欲就 ChatGPT 深度集成合作起诉 Apple 的内幕。核心逻辑：Apple 把 ChatGPT 塞进 iOS/Siri，不需要给 OpenAI 付一分钱座席费或分成，借此提升了「Apple Intelligence」的估值。但 OpenAI 没有拿到任何可转化的升级付费（用户留在系统界面里玩，根本不升级 ChatGPT Plus）。

Benioff 称为「There is a permanent abstraction layer that Apple controls. Apple owns the customer relationship. OpenAI performs the compute under total disadvantage as a free services company.」
这一案例深刻展示了：在 AI 时代，价值不是沉淀在「模型最聪明的人」那里，而是沉淀在「控制用户最末端屏幕的人（Apple OS、Google Android）」那里。这直接支撑了 Creekstone 「重资产端不碰，深挖行业特定 screen/harness 集成」的投资主旋律。
★★
Singularity
Demis Hassabis 爆料「Foothills of the Singularity」：医疗 AI 最快在 2 年内让传统十年的新药研发缩至 2 周
Rowan Cheung 关于 Demis 专访的解读：Hassabis 认为我们已经踩在了「奇点的山脚下」。他最兴奋的不仅是 AGI 2030，更是 AI 与生命科学的碰撞。Isomorphic 平台的 IsoDDE 声称可以通过模拟直接设计出能够有效抗病的分子，让平均长达 10 年的新药临床研发期在未来压缩至 2 周完成，彻底解决人类 90% 绝症。
AI for Science 正在从概念变成十亿美元级别的产业（Isomorphic本周 $2.1B 获得主权级参投）。这是大模型在「代码/文字」之外最确定的物理落地。

🏗️
Microsoft 7 In-House Models · 含推理级 MAI-Thinking-1
Microsoft AI Superintelligence Team · Build 2026 · 2026-06-02
基础模型大厂去O
微软在其 Build 2026 大会上推出全自研的 7 个基础模型家族，包括其首个自主推理模型：MAI-Thinking-1，旨在「Lessen reliance on OpenAI and lower costs for developers」——这是微软建立自研大模型生态、减低对 OpenAI 持股公司财务依赖的最明确表现。
MAI-Thinking-1: 推理性价比的新锚点

Yahoo Finance 的报道原话：

「According to Microsoft, MAI-Thinking-1 is a midsize AI model designed for high efficiency and low-token cost. The company said independent benchmarks place it within 5% of OpenAI's o1-preview on complex logical reasoning tasks while running at less than 1/4 of the inference cost and latency.」

这说明微软在走一条非常务实的「小参数高性能」中端路线。不盲目追求 GPT-5.4 级别的超大规模参数，而是用 midsize 加上微软的「多模型协同/符号约束」技术，实现 95% 推理质量下的超低价部署。

大厂 AI 的战略分野

以前微软是 OpenAI 的「算力提款机」，现在微软把 OpenAI 当作「普通备选 API 之一」。微软在 Build 上推行的是「多模型共存的 Azure Model Catalog」，自研模型家族 7 大金刚上线，让微软可以：

  • 大幅降低自营 Copilot 产品线（Office、Windows 等）的底层 token 运行结算成本；
  • 给开发者提供更便宜的自研模型选择，吸引价格敏感型团队；
  • 摆脱与 OpenAI 纠缠不清的 nonprofit/profit 内部控制风波。

对 AI 创业公司的影响：基础大模型本身正在彻底沦为「大厂的白电业务」（高 capex、极低利润率、为自置业务提效），应用层和垂直 Agent 的溢价红利进一步扩大。
